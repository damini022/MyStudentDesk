## Project Description

This project was created to understand how Python applications interact with databases and how a simple web interface can be built using Streamlit.  
The system allows users to add, view, update, and delete student records in real time.

---

## Features

- Add new student records  
- View all student details in tabular form  
- Update student marks using student ID  
- Delete student records  
- Web-based application deployed on Streamlit Cloud
